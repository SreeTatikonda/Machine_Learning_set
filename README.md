# Machine Learning Projects

A collection of machine learning projects demonstrating various ML techniques and real-world applications.

## Projects

### Email Classifier
Intelligent email categorization system that automatically sorts emails into 5 categories with confidence scores.

- Categories: Urgent, Spam, Newsletter, Work, Personal
- Accuracy: 100% on test dataset
- Model: Logistic Regression with TF-IDF features
- Interfaces: Web UI, REST API, CLI tool
- Tech Stack: Python, scikit-learn, Flask, NLP

View the project in the email-classifier branch or navigate to the email_classifier folder.

## Technologies Used

- TF-IDF Vectorization
